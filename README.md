<h2 id="running">Running the application 🎲</h2>
 
```bash
# Clone the repository
$ git clone https://github.com/HugoCortezL/credit-card-creation.git
 
# Go to folder
$ cd credit-card-creation
 
# Install the dependencies
$ npm install
 
# Start the application
$ npm run dev
```
